# GPT-2 Text Generation on Indian Cinema Dataset

## Overview

This project demonstrates fine-tuning GPT-2 for text generation using a custom Indian Cinema dataset. The model generates contextually relevant text related to Bollywood, Tollywood, film reviews, directors, actors, music, and cinema culture.

## Technologies Used

* Python
* Google Colab
* Hugging Face Transformers
* PyTorch
* GPT-2

## Project Workflow

1. Load GPT-2 pre-trained model.
2. Create and preprocess custom Indian Cinema dataset.
3. Tokenize the dataset.
4. Fine-tune GPT-2.
5. Generate text using prompts.
6. Evaluate generated outputs.

## Sample Prompt

Artificial Intelligence

## Sample Output

The model generates coherent text based on the learned patterns from the training dataset.

## Results

Successfully fine-tuned GPT-2 and generated contextually relevant text from custom prompts.
